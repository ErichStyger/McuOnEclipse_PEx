Component SSD1351
Display driver for the Solomon Systech SSD1351 display (e.g. on Hexiwear)
Component Level: Low
Category: CPU External Devices-Display
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • GetWidth - Returns the width of the display in pixels (in x direction)

      ANSIC prototype: ComponentName_PixelDim GetWidth(void)
    • Return value:ComponentName_PixelDim - Error code

  • GetHeight - Returns the height of the display in pixels (in y direction)

      ANSIC prototype: ComponentName_PixelDim GetHeight(void)
    • Return value:ComponentName_PixelDim - Error code

  • GetLongerSide - Returns the size of the longer side of the display

      ANSIC prototype: ComponentName_PixelDim GetLongerSide(void)
    • Return value:ComponentName_PixelDim - Error code

  • GetShorterSide - Returns the size of the shorter side of the display

      ANSIC prototype: ComponentName_PixelDim GetShorterSide(void)
    • Return value:ComponentName_PixelDim - Error code

  • SetDisplayOrientation - Sets the display orientation. If you enable this method, then the orientation of the display can be changed at runtime. However, this requires additional resources.

      ANSIC prototype: void SetDisplayOrientation(ComponentName_DisplayOrientation newOrientation)
    • newOrientation:ComponentName_DisplayOrientation - new orientation to be used

  • GetDisplayOrientation - Returns the current display orientation

      ANSIC prototype: ComponentName_DisplayOrientation GetDisplayOrientation(void)
    • Return value:ComponentName_DisplayOrientation - current display orientation

  • ReadDataWord - Writes a single word to the bus

      ANSIC prototype: word ReadDataWord(void)
    • Return value:word - data word from the display

  • WriteData - Writes a data byte to the bus

      ANSIC prototype: void WriteData(uint8_t data)
    • data:uint8_t - data byte to send

  • WriteDataWord - Sends a data word to the display

      ANSIC prototype: void WriteDataWord(uint16_t data)
    • data:uint16_t - data to write

  • WriteDataWordRepeated - Sends a data word to the display a number of times

      ANSIC prototype: void WriteDataWordRepeated(uint16_t data, size_t nof)
    • data:uint16_t - data to write
    • nof:byte - How many times the data word shall be sent

  • WriteDataBlock - Sends a data buffer to the display

      ANSIC prototype: void WriteDataBlock(uint8_t *data, size_t dataSize)
    • data: Pointer to uint8_t - Pointer to data to write
    • dataSize:byte - no hint

  • WriteCommand - Sends a command byte to the bus

      ANSIC prototype: void WriteCommand(uint8_t cmd)
    • cmd:uint8_t - the command to be sent

  • OpenWindow - Opens a window inside the display we want to write to.

      ANSIC prototype: void OpenWindow(ComponentName_PixelDim x0, ComponentName_PixelDim y0, ComponentName_PixelDim x1, ComponentName_PixelDim y1)
    • x0:ComponentName_PixelDim - no hint
    • y0:ComponentName_PixelDim - no hint
    • x1:ComponentName_PixelDim - no hint
    • y1:ComponentName_PixelDim - no hint

  • CloseWindow - Closes a window previously opened with OpenWindow()

      ANSIC prototype: void CloseWindow(void)

  • ClearBuffer - Clears the display buffer

      ANSIC prototype: void ClearBuffer(ComponentName_PixelColor color)
    • color:ComponentName_PixelColor - color used to clear the buffer

  • Clear - Clears the whole display memory.

      ANSIC prototype: void Clear(void)

  • UpdateFull - Updates the whole display. This is only a stub for this display as we are using windowing.

      ANSIC prototype: void UpdateFull(void)

  • UpdateRegion - Updates a region of the display. This is only a stub for this display as we are using windowing.

      ANSIC prototype: void UpdateRegion(ComponentName_PixelDim x, ComponentName_PixelDim y, ComponentName_PixelDim w, ComponentName_PixelDim h)
    • x:ComponentName_PixelDim - x coordinate
    • y:ComponentName_PixelDim - y coordinate
    • w:ComponentName_PixelDim - width of the region
    • h:ComponentName_PixelDim - Height of the region

  • InitCommChannel - Method to initialize communication to the LCD. Needed if the bus to the LCD is shared with other components and settings are different.

      ANSIC prototype: void InitCommChannel(void)

  • OnDataReceived - callback to be called from SPI interrupt

      ANSIC prototype: void OnDataReceived(void)

  • PutPixel - Writes a pixel to the display memory buffer

      ANSIC prototype: void PutPixel(ComponentName_PixelDim x, ComponentName_PixelDim y, ComponentName_PixelColor color)
    • x:ComponentName_PixelDim - x coordinate
    • y:ComponentName_PixelDim - no hint
    • color:ComponentName_PixelColor - pixel color

  • GetPixel - Returns a pixel from the memory buffer

      ANSIC prototype: ComponentName_PixelColor GetPixel(ComponentName_PixelDim x, ComponentName_PixelDim y)
    • x:ComponentName_PixelDim - x coordinate
    • y:ComponentName_PixelDim - y coordinate
    • Return value:ComponentName_PixelColor - Error code

  • DisplayOnOff - Turns the display on or off

      ANSIC prototype: uint8_t DisplayOnOff(bool on)
    • on:bool - TRUE to turn the display on, FALSE to turn it off
    • Return value:uint8_t - Error code

  • SetContrastABC - Sets the contrast ABC value (Command 0xC1)

      ANSIC prototype: uint8_t SetContrastABC(uint8_t contrastA, uint8_t contrastB, uint8_t contrastC)
    • contrastA:uint8_t - contrast A value
    • contrastB:uint8_t - contrast B value
    • contrastC:uint8_t - contrast C value
    • Return value:uint8_t - Error code

  • SetContrastMaster - Sets the contrast Master value (Command 0xC7)

      ANSIC prototype: uint8_t SetContrastMaster(uint8_t contrast)
    • contrast:uint8_t - contrast master value
    • Return value:uint8_t - Error code

  • ParseCommand - Shell Command Line parser. Method is only available if Shell is enabled in the component properties.

      ANSIC prototype: uint8_t ParseCommand(const unsigned char* cmd, bool *handled, const %@Shell@'ModuleName'%.StdIOType *io)
    • cmd:byte - command string
    • handled: Pointer to bool - Pointer to variable which tells if the command has been handled or not
    • io:byte - Pointer to I/O structure
    • Return value:uint8_t - Error code

  • GetLCD - Method to be called for mutual exclusive access to the LCD

      ANSIC prototype: void GetLCD(void)

  • GiveLCD - Method to be called for mutual exclusive access to the LCD

      ANSIC prototype: void GiveLCD(void)

  • GetBus - Called to get the SPI bus

      ANSIC prototype: void GetBus(void)

  • Deinit - Driver de-initialization

      ANSIC prototype: void Deinit(void)

  • Init - Display driver initialization

      ANSIC prototype: void Init(void)
