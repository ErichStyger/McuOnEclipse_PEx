Embedded Components
Component OneWire
Component for the One Wire Protocol
Component Level: High
Category: Software-Communication
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component Name - Name of the component.
  • Data Pin I/O - Link component (for details about settings see <a href="../../DOCs/InheritanceAndSharing.html">Component Inheritance &amp; Component Sharing</a>).
  • TimerUnit - Link component (for details about settings see <a href="../../DOCs/InheritanceAndSharing.html">Component Inheritance &amp; Component Sharing</a>).
  • Connection time settings -
    • Connection Mode

      There are 2 options:
      • Master - One slave: Master, one slave
      • Master - Multiple devices: Master, multiple devices

    • Reset time (us) Duration of the reset pulse that the master send to slave devices.
    • Device response time (us) Time that master waits before read, after send the reset pulse.
    • High signal time (us) Period that master holds the DQ signal low to send a bit '1'.
    • Low signal time (us) Period that master holds the DQ signal low to send a bit '0'.
    • Output low time to read (us)
    • Wait time to read input (us) Wait time to read after start to send the low signal to read.
    • Total slot time (us)
  • Buffers -
    • Output buffer size Determines the output buffer size.
    • Input buffer size
    • Time buffer size
    • Program buffer size
  • CriticalSection - Link component (for details about settings see <a href="../../DOCs/InheritanceAndSharing.html">Component Inheritance &amp; Component Sharing</a>).
  • Use pre-charged Timer_Unit configuration -
