Embedded Components
Component OneWire
Component for the One Wire Protocol
Component Level: High
Category: Software-Communication
Events:
(Events are call-back functions called when an important event occurs. Please see the Embedded Components page for more information.)
  • EventModule - Name of the user module (without extension), where the events from this component are placed.
  • Event module name -

      ANSIC prototype:void Event module name

  • OnProgramEvent -

      ANSIC prototype:void OnProgramEvent(uint8_t key4)
    • key4:uint8_t - Key to identify the source of the event

  • OnError -

      ANSIC prototype:void OnError(void* error)
    • error:void* -

  • OnBlockReceived -

      ANSIC prototype:void OnBlockReceived(void)

  • OnWaitEnd -

      ANSIC prototype:void OnWaitEnd(uint8_t key2)
    • key2:uint8_t - Key to identificate the source of the event.

  • OnSendEnd -

      ANSIC prototype:void OnSendEnd(void)

  • OnSendedReset -

      ANSIC prototype:void OnSendedReset(void)

For each generated event it is possible to set:
  • Procedure Name - name of procedure (written by user) that handles the event.
