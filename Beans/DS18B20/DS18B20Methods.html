Embedded Components
Component DS18B20
Driver to read the temperature from DS18B20 sensor, using a TimerUnit to control the pulse generation.
Component Level: High
Category: Software-Communication
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • Init - Initializes the device.

      ANSIC prototype: void Init(void)
    • Return value:void -

  • ReadRom - Starts to read the rom code and saves it in memory.

      ANSIC prototype: bool ReadRom(void* sensor_index)
    • sensor_index:void* -
    • Return value:bool - Returns FALSE if the device is busy in another operation.

  • GetRomCode - Gets the rom code from the memory.

      ANSIC prototype: char* GetRomCode(void* sensor_index)
    • sensor_index:void* -
    • Return value:char* - Returns a pointer to the rom code array.
