Component LEDmatrix
LED Matrix Display using row/column access
Component Level: Low
Category: CPU External Devices-Display
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • GetWidth - Returns the width of the display in pixels/LED's

      ANSIC prototype: ComponentName_PixelDim GetWidth(void)
    • Return value:ComponentName_PixelDim - width of the display

  • GetHeight - Returns the height of the display in pixels/LED's

      ANSIC prototype: ComponentName_PixelDim GetHeight(void)
    • Return value:ComponentName_PixelDim - height of the display

  • GetShorterSide - Returns the size of the display shorter side

      ANSIC prototype: ComponentName_PixelDim GetShorterSide(void)
    • Return value:ComponentName_PixelDim - shorter side of the display

  • GetLongerSide - Returns the size of the display longer side

      ANSIC prototype: ComponentName_PixelDim GetLongerSide(void)
    • Return value:ComponentName_PixelDim - longer side of the display

  • GetDisplayOrientation - Returns the current display orientation

      ANSIC prototype: ComponentName_DisplayOrientation GetDisplayOrientation(void)
    • Return value:ComponentName_DisplayOrientation - current display orientation

  • SetDisplayOrientation - Returns the current display orientation

      ANSIC prototype: void SetDisplayOrientation(ComponentName_DisplayOrientation newOrientation)
    • newOrientation:ComponentName_DisplayOrientation - no hint

  • StartSafeReverseBias - Puts the LEDs in reverse bias mode and starts discharing (port set as input).

      ANSIC prototype: void StartSafeReverseBias(void *portBackup)
    • portBackup: Pointer to byte - Pointer to backup structure for port status (to be restored later). It points to a %'ModuleName'%.PortStatus structure to be allocated by the caller.

  • GetCathodeVal - Returns the cathode value

      ANSIC prototype: ComponentName_SensorValType GetCathodeVal(void)
    • Return value:ComponentName_SensorValType - The value of the cathode while discharging in reverse bias mode.

  • StopSafeReverseBias - Reverts the port status back to previous mode.

      ANSIC prototype: void StopSafeReverseBias(void *portBackup)
    • portBackup: Pointer to byte - Pointer to backup structure for port status. It points to a %'ModuleName'%.PortStatus structure to be allocated by the caller.

  • UpdateFull - Updates the full display.

      ANSIC prototype: void UpdateFull(void)

  • UpdateRegion - Updates a region of the display

      ANSIC prototype: void UpdateRegion(ComponentName_PixelDim x, ComponentName_PixelDim y, ComponentName_PixelDim w, ComponentName_PixelDim h)
    • x:ComponentName_PixelDim - x coordinate
    • y:ComponentName_PixelDim - y coordinate
    • w:ComponentName_PixelDim - width of region
    • h:ComponentName_PixelDim - height of region

  • ClearBuffer - Clears the data buffer

      ANSIC prototype: void ClearBuffer(byte color)
    • color:byte - Color used for clearing

  • On - Turns the LEDs on.

      ANSIC prototype: void On(void)

  • Off - Turns the LEDs off.

      ANSIC prototype: void Off(void)

  • Enable - Enables showing the content of the display.

      ANSIC prototype: void Enable(void)

  • Disable - Disables showing the content of the display.

      ANSIC prototype: void Disable(void)

  • GetLCD - Method to be called for mutual exclusive access to the display

      ANSIC prototype: void GetLCD(void)

  • GiveLCD - Method to be called to release mutual exclusive access to the display

      ANSIC prototype: void GiveLCD(void)


Note: Some methods can be implemented as macros.
