Component PDC8544
Display driver for the Philips PDC8544 display driver
Component Level: Low
Category: CPU External Devices-Display
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • GetWidth - Returns the width of the display in pixels

      ANSIC prototype: ComponentName_PixelDim GetWidth(void)
    • Return value:ComponentName_PixelDim - width of display

  • GetHeight - Returns the height of the display in pixels

      ANSIC prototype: ComponentName_PixelDim GetHeight(void)
    • Return value:ComponentName_PixelDim - height of display

  • GetLongerSide - Returns the longer side of the display in pixels

      ANSIC prototype: ComponentName_PixelDim GetLongerSide(void)
    • Return value:ComponentName_PixelDim - longer side of display

  • GetShorterSide - Returns the shorter side of the display in pixels

      ANSIC prototype: ComponentName_PixelDim GetShorterSide(void)
    • Return value:ComponentName_PixelDim - shorter side of display

  • SetDisplayOrientation - Sets current display orientation

      ANSIC prototype: void SetDisplayOrientation(ComponentName_DisplayOrientation newOrientation)
    • newOrientation:ComponentName_DisplayOrientation - the new orientation for the display

  • GetDisplayOrientation - Returns the current display orientation

      ANSIC prototype: ComponentName_DisplayOrientation GetDisplayOrientation(void)
    • Return value:ComponentName_DisplayOrientation - current display orientation

  • SetPos - Sets the cursor to the given position

      ANSIC prototype: void SetPos(ComponentName_PixelDim x, ComponentName_PixelDim y)
    • x:ComponentName_PixelDim - The column number (in bits). In the range of 0...83.
    • y:ComponentName_PixelDim - The row number (byte rows), in the range of 0..5.

  • ClearBuffer - Writes a color to the full display buffer and clears it.

      ANSIC prototype: void ClearBuffer(ComponentName_PixelColor color)
    • color:ComponentName_PixelColor - Pixel color value

  • PutPixel - Writes a pixel to the display RAM buffer

      ANSIC prototype: void PutPixel(ComponentName_PixelDim x, ComponentName_PixelDim y, ComponentName_PixelColor color)
    • x:ComponentName_PixelDim - Pixel x coordinate
    • y:ComponentName_PixelDim - Pixel y coordinate
    • color:ComponentName_PixelColor - Pixel color value

  • Clear - Clears the whole display memory.

      ANSIC prototype: void Clear(void)

  • ClearLine - Clears a character line on the display

      ANSIC prototype: void ClearLine(uint8_t lineNumber)
    • lineNumber:uint8_t - the line number of the line to be cleared.

  • WriteChar - Writes a single character to the display at the current position.

      ANSIC prototype: void WriteChar(uint8_t ch)
    • ch:uint8_t - Char to be written to the display.

  • WriteString - Writes a zero byte terminated string to the display at the current cursor position.

      ANSIC prototype: void WriteString(uint8_t *str)
    • str: Pointer to uint8_t - Pointer to string to be written to the display.

  • WriteLineStr - Writes a string to the given line position.

      ANSIC prototype: void WriteLineStr(uint8_t line, uint8_t *str)
    • line:uint8_t - The line number, starting with 0.
    • str: Pointer to uint8_t - Pointer to zero byte terminated string

  • SetContrast - Sets the display contrast.

      ANSIC prototype: void SetContrast(uint8_t contrast)
    • contrast:uint8_t - Contrast value, must be in the range of 0 >= x < 128

  • SetMode - Sets the mode of the display (if inverted or not).

      ANSIC prototype: void SetMode(bool inverted)
    • inverted:bool - Pass TRUE to have the display operate in inverted mode, FALSE for normal mode.

  • SetWindow - Defines a window inside the display we want to write to.

      ANSIC prototype: void SetWindow(ComponentName_PixelDim x0, ComponentName_PixelDim x1, ComponentName_PixelDim y0, ComponentName_PixelDim y1)
    • x0:ComponentName_PixelDim - no hint
    • x1:ComponentName_PixelDim - no hint
    • y0:ComponentName_PixelDim - no hint
    • y1:ComponentName_PixelDim - no hint

  • UpdateFull - Updates the whole display

      ANSIC prototype: void UpdateFull(void)

  • UpdateRegion - Updates a region of the display

      ANSIC prototype: void UpdateRegion(ComponentName_PixelDim x, ComponentName_PixelDim y, ComponentName_PixelDim w, ComponentName_PixelDim h)
    • x:ComponentName_PixelDim - x coordinate
    • y:ComponentName_PixelDim - y coordinate
    • w:ComponentName_PixelDim - width of the region
    • h:ComponentName_PixelDim - height of the region

  • GetLCD - Method to be called for mutual exclusive access to the LCD

      ANSIC prototype: void GetLCD(void)

  • GiveLCD - Method to be called for mutual exclusive access to the LCD

      ANSIC prototype: void GiveLCD(void)

  • Init - Display driver initialization

      ANSIC prototype: void Init(void)

  • Deinit - Module Deinitialization

      ANSIC prototype: void Deinit(void)
