Component minIni
minIni is a programmer’s library to read and write 'ini' files in embedded systems.
Component Level: High
Category: SW-User Components
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • ini_getbool - Return a boolean key. A true boolean is found if one of the following is matched:
    - A string starting with 'y' or 'Y'
    - A string starting with 't' or 'T'
    - A string starting with '1'
    A false boolean is found if one of the following is matched:
    - A string starting with 'n' or 'N'
    - A string starting with 'f' or 'F'
    - A string starting with '0'

      ANSIC prototype: int ini_getbool(const mTCHAR *Section, const mTCHAR *Key, int DefValue, const mTCHAR *Filename)
    • Section: Pointer to char - The name of the section to search for
    • Key: Pointer to char - The name of the entry to find the value of
    • DefValue:int - Default value in the event of a failed read; it should be zero (0) or one (1).
    • Filename: Pointer to char - The name and full path of the .ini file to read from
    • Return value:int - The true/false flag as interpreted at Key

  • ini_gets - Return the string for a given section and key.

      ANSIC prototype: int ini_gets(const mTCHAR *Section, const mTCHAR *Key, const mTCHAR *DefValue, nTCHAR *Buffer, int BufferSize, const mTCHAR *Filename)
    • Section: Pointer to uint8_t - The name of the section to search for
    • Key: Pointer to char - Pointer to the name of the entry to find the value of
    • DefValue: Pointer to char - Pointer to the default value in the event of a failed read
    • Buffer: Pointer to uint8_t - A pointer to the buffer to copy into
    • BufferSize:int - The maximum number of characters to copy
    • Filename: Pointer to char - The name and full path of the .ini file to read from
    • Return value:int - The number of characters copied into the supplied buffer

  • ini_puts - Writes a string value for a given section and key. Method is only enabled if 'Read only' is set to 'no' in the properties.

      ANSIC prototype: int ini_puts(const mTCHAR *Section, const mTCHAR *Key, const mTCHAR *Value, const mTCHAR *Filename)
    • Section: Pointer to uint8_t - The name of the section to write the string in
    • Key: Pointer to char - The name of the entry to write
    • Value: Pointer to char - Pointer to the buffer the string, or NULL to erase the key
    • Filename: Pointer to char - The name and full path of the .ini file to write to
    • Return value:int - 1 if successful, otherwise 0

  • ini_getl - Return the long integral value for a given section and key.

      ANSIC prototype: long ini_getl(const mTCHAR *Section, const mTCHAR *Key, long DefValue, const mTCHAR *Filename)
    • Section: Pointer to byte - The name of the section to search for
    • Key: Pointer to char - Pointer to the name of the entry to find the value of
    • DefValue:long - Default value in the event of a failed read
    • Filename: Pointer to char - The name and full path of the .ini file to read from
    • Return value:long - The value located at Key

  • ini_putl - Writes a long integral value for a given section and key. Method is only enabled if 'Read only' is set to 'no' in the properties.

      ANSIC prototype: int ini_putl(const mTCHAR *Section, const mTCHAR *Key, long Value, const mTCHAR *Filename)
    • Section: Pointer to byte - The name of the section to write the value in
    • Key: Pointer to char - The name of the entry to write
    • Value:long - The value to write
    • Filename: Pointer to char - The name and full path of the .ini file to write to
    • Return value:int - 1 if successful, otherwise 0

  • ini_getf - Return the long integral value for a given section and key. Only enabled if 'Use Real' is enabled in properties.

      ANSIC prototype: INI_REAL ini_getf(const mTCHAR *Section, const mTCHAR *Key, INI_REAL DefValue, const mTCHAR *Filename)
    • Section: Pointer to byte - The name of the section to search for
    • Key: Pointer to char - Pointer to the name of the entry to find the value of
    • DefValue:INI_REAL - Default value in the event of a failed read
    • Filename: Pointer to char - The name and full path of the .ini file to read from
    • Return value:INI_REAL - The value located at Key

  • ini_putf - Writes a long float value for a given section and key. Method is only enabled if 'Read only' is set to 'no' in the properties and 'Use Real' is enabled.

      ANSIC prototype: int ini_putf(const mTCHAR *Section, const mTCHAR *Key, INI_REAL Value, const mTCHAR *Filename)
    • Section: Pointer to byte - The name of the section to write the value in
    • Key: Pointer to char - The name of the entry to write
    • Value:INI_REAL - The value to write
    • Filename: Pointer to char - The name and full path of the .ini file to write to
    • Return value:int - 1 if successful, otherwise 0

  • ini_hassection - Used to find out if section exists. Returns 1 if the section has been found, 0 otherwise.

      ANSIC prototype: int ini_hassection(const mTCHAR *Section, const mTCHAR *Filename)
    • Section: Pointer to byte - The name of the section
    • Filename: Pointer to char - The name and full path of the .ini file to read from
    • Return value:int - The number of characters copied into the supplied buffer

  • ini_getsection - Return the key inside a section

      ANSIC prototype: int ini_getsection(int idx, mTCHAR *Buffer, int BufferSize, const mTCHAR *Filename)

    • Buffer: Pointer to char - A pointer to the buffer to copy into

    • Filename: Pointer to char - The name and full path of the .ini file to read from
    • Return value:int - The number of characters copied into the supplied buffer

  • ini_haskey - Used to find if a key exists. Returns 1 if key has been found, 0 otherwise.

      ANSIC prototype: int ini_haskey(const mTCHAR *Section, const mTCHAR *Key, const mTCHAR *Filename)
    • Section: Pointer to byte - The name of the section
    • Key: Pointer to char - The name of the entry to find
    • Filename: Pointer to char - The name and full path of the .ini file
    • Return value:int - 1 if successful, otherwise 0

  • ini_getkey - Return the key inside a section

      ANSIC prototype: int ini_getkey(const mTCHAR *Section, int idx, mTCHAR *Buffer, int BufferSize, const mTCHAR *Filename)
    • Section: Pointer to byte - The name of the section to browse through, or NULL to browse through the keys outside any section
    • idx:int - The zero-based sequence number of the key to return
    • Buffer: Pointer to char - A pointer to the buffer to copy into
    • BufferSize:int - The maximum number of characters to copy
    • Filename: Pointer to char - The name and full path of the .ini file to read from
    • Return value:int - The number of characters copied into the supplied buffer

  • ini_browse - Function to browse the ini file. With this function the file is opened only once.

      ANSIC prototype: int ini_browse(INI_CALLBACK Callback, const void *UserData, const TCHAR *Filename)
    • Callback:uint8_t - a pointer to a function that will be called for every setting in the INI file.
    • UserData:uint8_t - arbitrary data, which the function passes on the the Callback function
    • Filename:uint8_t - the name and full path of the .ini file to read from
    • Return value:int - 1 on success, 0 on failure (INI file not found)
