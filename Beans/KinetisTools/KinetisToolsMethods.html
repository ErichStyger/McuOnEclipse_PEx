Component KinetisTools
Component supporting various functions of Kinetis ARM Cortex devices
Component Level:
Category: SW-User Components
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • SoftwareReset - Performs a reset of the device

      ANSIC prototype: void SoftwareReset(void)

  • UIDGet - Return the 128bit UID of the device

      ANSIC prototype: uint8_t UIDGet(ComponentName_UID *uid)
    • uid: Pointer to byte - Pointer to
    • Return value:uint8_t - Error code

  • UIDSame - Compares two UID

      ANSIC prototype: bool UIDSame(const ComponentName_UID *src, const ComponentName_UID *dst)
    • src: Pointer to byte - Pointer to
    • Variable_1:byte - no hint
    • Return value:bool - TRUE if the same, FALSE otherwise

  • UIDtoString - Returns the value of the UID as string

      ANSIC prototype: uint8_t UIDtoString(const ComponentName_UID *uid, uint8_t *buf, size_t bufSize)
    • uid:byte - no hint
    • buf: Pointer to uint8_t - Pointer to
    • bufSize:word - no hint
    • Return value:uint8_t - Error code

  • GetKinetisFamilyString - Determines the Kinetis Familiy based on SIM_SDID register

      ANSIC prototype: ComponentName_ConstCharPtr GetKinetisFamilyString(void)
    • Return value:ComponentName_ConstCharPtr - String describing the Kinetis Family

  • GetPC - returns the program counter

      ANSIC prototype: void* GetPC(void)
    • Return value:void* - program counter

  • GetSP - returns the stack pointer

      ANSIC prototype: void* GetSP(void)
    • Return value:void* - stack pointer

  • SetPSP - sets the process stack pointer

      ANSIC prototype: void SetPSP(void *setval)
    • setval:byte - new PSP value

  • SetLR - Sets the link register

      ANSIC prototype: void SetLR(uint32_t setval)
    • setval:uint32_t - new LR value

  • InitCycleCounter - Initializes the cycle counter, available if the core has a DWT (Data Watchpoint and Trace) unit, usually present on M3/M4/M7

      ANSIC prototype: void InitCycleCounter(void)

  • ResetCycleCounter - Reset the cycle counter (set it to zero)

      ANSIC prototype: void ResetCycleCounter(void)

  • EnableCycleCounter - Enables counting the cycles.

      ANSIC prototype: void EnableCycleCounter(void)

  • DisableCycleCounter - Disables the cycle counter.

      ANSIC prototype: void DisableCycleCounter(void)

  • GetCycleCounter - Return the current cycle counter value

      ANSIC prototype: uint32_t GetCycleCounter(void)
    • Return value:uint32_t - cycle counter

  • GetUsedMainStackSpace - Returns the used main stack space, based on the overwritten checking pattern.

      ANSIC prototype: dword GetUsedMainStackSpace(void)
    • Return value:dword - Error code

  • GetUnusedMainStackSpace - Calculates the unused stack space, based on the checking pattern.

      ANSIC prototype: dword GetUnusedMainStackSpace(void)
    • Return value:dword - Error code

  • FillMainStackSpace - Fill the stack space with the checking pattern, up to the current MSP.

      ANSIC prototype: void FillMainStackSpace(void)

  • GetLinkerMainStackSize - Returns the size of the main (MSP) stack size, using linker symbols for top (higher address) and base (lower address).

      ANSIC prototype: dword GetLinkerMainStackSize(void)
    • Return value:dword - Error code

  • GetLinkerMainStackTop - Return the stack top, as set in the linker file. The stack grows from the top (higher address) to the base (lower address).

      ANSIC prototype: void* GetLinkerMainStackTop(void)
    • Return value:void* - Error code

  • GetLinkerMainStackBase - Return the stack bottom, as configured in the linker file. The stack grows from the top (higher address) to the base (lower address).

      ANSIC prototype: void* GetLinkerMainStackBase(void)
    • Return value:void* - Error code

  • ParseCommand - Shell Command Line parser. Method is only available if Shell is enabled in the component properties.

      ANSIC prototype: uint8_t ParseCommand(const unsigned char* cmd, bool *handled, const %@Shell@'ModuleName'%.StdIOType *io)
    • cmd:byte - Pointer to command string
    • handled: Pointer to bool - Pointer to variable which tells if the command has been handled or not
    • io: Pointer to byte - Pointer to I/O structure
    • Return value:uint8_t - Error code

  • Deinit - Driver de-initialization routine

      ANSIC prototype: void Deinit(void)

  • Init - Driver initialization routine

      ANSIC prototype: void Init(void)
